# Prism Quest MVP 요구사항

## 사용자
- 아이: 미션 완료 요청, 포인트와 보상 확인
- 부모: 완료 승인, 포인트 조정, 보상 사용 승인

## 핵심 규칙
- 미션 체크만으로 포인트를 지급하지 않는다.
- 부모 승인 후 포인트를 지급한다.
- 점수를 깎는 벌점보다 미획득을 기본으로 한다.
- 모든 포인트 변경은 원장(ledger)에 기록한다.

## 1차 범위
1. 오늘의 미션 목록
2. 완료 요청
3. 부모 PIN 승인
4. 포인트 원장
5. 보상 사용 요청 및 승인
6. 모바일 홈 화면 추가가 가능한 PWA
