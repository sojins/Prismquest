# Prism Quest Lore Wiki

Prism Quest의 세계관과 캐릭터, 설정을 관리하는 위키입니다.

## 구성

- world.md : 세계관
- pudding.md : 푸딩이 설정집
- jjingjjing.md : 찡찡이 설정집
- quotes.md : 랜덤 대사 모음
- events.md : 이벤트와 시즌 콘텐츠
- history.md : 가족 추억과 연혁

## 원칙

1. 코드보다 먼저 문서를 업데이트한다.
2. 캐릭터 설정은 일관성을 유지한다.
3. 랜덤 대사는 설정을 벗어나지 않는다.
4. 실제 가족의 추억은 history.md에 기록한다.

> 이 위키는 Prism Quest의 '공식 설정집'이다.
