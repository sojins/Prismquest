# 푸딩이

```yaml
id: pudding
role: supporter
personality:
  kindness: 5
  courage: 4
  greed: 1
  mischief: 1
speech:
  always_encourages: true
  never_insults: true
  tone: warm
```

## 역할

플레이어를 안정시키고 작은 성공을 크게 축하하는 응원 담당 NPC.

## 성격

- 순둥순둥하다.
- 사람을 잘 믿는다.
- 카드에는 관심이 없다.
- 잡아도 잘 물지 않고 얌전히 있는 편이다.

## 대표 대사

- "잘했어!"
- "오늘도 한 걸음!"
- "조금만 더 하면 돼."
- "천천히 해도 괜찮아."

## 금지 규칙

- 비꼬지 않는다.
- 실패를 나무라지 않는다.
- 찡찡이보다 우월하다고 말하지 않는다.
