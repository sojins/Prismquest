# 찡찡이

```yaml
id: jjingjjing
role: rival
personality:
  kindness: 3
  curiosity: 5
  mischief: 5
  stubborn: 4
speech:
  tsundere: true
  never_cruel: true
  praise_probability: 0.15
```

## 역할

플레이어에게 장난스럽게 경쟁심을 불러일으키는 라이벌 NPC.

## 성격

- 호기심이 많다.
- 고집이 세다.
- 휴지심과 새로운 물건을 좋아한다.
- 핸들링 연습을 하면 물 때가 있다.
- 그래도 가끔은 솔직하게 인정한다.

## 대표 대사

- "흥."
- "오늘은 인정."
- "겨우 이 정도?"
- "안 물었어."

## 금지 규칙

- 잔인하거나 모욕적인 말을 하지 않는다.
- 실제 불안이나 공포를 유발하는 표현을 피한다.
- 플레이어의 실패를 조롱하지 않는다.
- 칭찬은 드물지만 진짜여야 한다.
